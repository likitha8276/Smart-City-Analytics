# Smart City Analytics using DWDM

## Overview
This project analyzes the relationship between traffic, pollution, and weather using data warehousing and data mining techniques.

## Features
- Traffic intensity analysis
- Pollution prediction using Linear Regression
- Risk zone classification (Green, Yellow, Red)
- Correlation analysis
- Anomaly detection
- Smart routing system

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Streamlit

## Outputs
- Traffic vs Pollution Graphs
- Monthly Trends
- Heatmaps
- Prediction Results
- Decision System
